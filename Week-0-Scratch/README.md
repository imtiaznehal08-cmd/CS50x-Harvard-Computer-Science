The Phishing Wolf and The Cybersec Goat

A Cybersecurity Educational Simulation

Description
The Phishing Wolf and The Cybersec Goat is a narrative-driven interactive simulation developed as a week 0 project for Harvard's CS50 Introduction to Computer Science. Built using the Scratch programming environment, the project serves as an educational tool to teach users about the psychological and technical mechanics of phishing attacks.

The project features a dual-perspective story where the "Phishing Wolf" attempts various social engineering tactics to compromise the "Cybersec Goat (Billy)." Users are tasked with identifying red flags and navigating proctored security scenarios.

Features
Interactive Decision Making: Players must choose how to respond to suspicious prompts and "Allow" or "Deny" access using custom UI sprites.

Technical Concept Integration: Teaches fundamental networking concepts, including the distinction between secure and insecure connections.

Asset Design: Includes custom-built sprites for a "Button Allow" interface and specialized phishing simulation imagery.

Educational Feedback: Immediate consequences for choices to demonstrate the impact of cybersecurity lapses.

Project Structure
The Phishing Wolf And The Cybersec Goat.sb3: The complete Scratch project file containing all scripts, costumes, and sounds.

README.md: This file, providing an overview of the project and its context within CS50.

Logic Implementation: The project utilizes broadcast messaging for scene transitions and complex variable tracking to manage user progress and security scores.

Cybersecurity Focus
Inspired by studies into professional certifications like CompTIA Security+, this project translates high-level concepts (like proctored environments and networking) into a digestible format suitable for a general audience.

How to Run
Navigate to the Scratch MIT Editor.

Select File > Load from your computer.

Upload the .sb3 file included in this repository.

Click the Green Flag to start the simulation.
